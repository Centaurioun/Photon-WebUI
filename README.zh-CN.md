# Photon WebUI

*Photon WebUI* 是一款管理 [aria2](https://github.com/aria2/aria2) 的网页前端。

如果你想要一个桌面下载器，请尝试 [*Photon*](https://github.com/alanzhangzm/Photon/blob/master/README.zh-CN.md)。

# 特色
- 管理多个 aria2 服务器
- BT：支持选择文件下载
